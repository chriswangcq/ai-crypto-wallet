# 🎮 AI Crypto Wallet - Examples

Ready-to-use examples for integrating AI Crypto Wallet into your projects.

## Quick Start

### Basic Usage
```bash
python examples/basic_usage.py
```
Check balances and addresses across all supported chains.

### LangChain Integration
```bash
export OPENAI_API_KEY=your_key
python examples/langchain_integration.py
```
Give your LangChain agent crypto capabilities.

### AutoGPT Plugin
Copy `autogpt_plugin.py` to your AutoGPT plugins directory.

## Use Cases

1. **AI-as-a-Service** - Let your AI accept crypto payments
2. **Autonomous Agents** - Self-funding AI that pays for its own resources
3. **Multi-Agent Economies** - Agents transacting with each other
4. **DeFi Automation** - AI managing crypto portfolios

## Custom Integrations

Want to integrate with a different framework? 
Check the source - the `AICryptoWallet` class provides a universal JSON interface that works with any system.
